Grid

The Material Design Lite (MDL) grid component is a simplified method for laying out content for multiple screen sizes.

Using the mdl-cell--N-col classes you can create a responsive grid.

Example: Device specific columns

In addition to the normal grid classes you can define which size a column should be for a specific device.

6 (8 tablet)
4 (6 tablet)
2 (4 phone)

Example: Hide columns

It is also possible to hide columns for a specific device.

.mdl-cell--hide-desktop
.mdl-cell--hide-tablet
.mdl-cell--hide-phone

Example: Align columns

Align columns within their parent column.

.mdl-cell--top
.mdl-cell--middle
.mdl-cell--bottom
